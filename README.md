## О проекте

Мобильное приложение, реализованное в рамках тестового задания для Авито.
AvitoBooks позволяет загружать собственные книги в облако, хранить локальные копии на устройстве и читать их в удобном интерфейсе с настраиваемой темой оформления.
Приложение демонстрирует работу с Firebase и современный, аккуратный UI на Jetpack Compose.

---

## Основной функционал

- **Аутентификация**
  - Вход и регистрация по email/паролю (Firebase Authentication).

- **Профиль пользователя**
  - Никнейм и статус с сохранением в **Firestore Database**.
  - Аватарка с загрузкой изображения и хранением в **Firebase Storage**.

- **Библиотека книг**
  - Загрузка файлов книг в облако (**PDF**, **EPUB**, **TXT**).
  - Хранение метаданных книг (название, автор, пути к файлам) в Firestore.
  - Скачивание книги на устройство в каталог, привязанный к `uid` пользователя.

- **Экран чтения**
  - Поддержка форматов **PDF / EPUB / TXT**.
  - Отображение и обновление прогресса чтения.

- **Backend-сервисы:**

  - Firestore Database
  - Firebase Storage
